### 사용법 ###


step 0. Creon Plus에 로그인되어 있어야 함!


step 1. DB 경로 입력 후 [연결] 버튼 클릭

- db 파일이 새 파일인 경우 분봉/일봉 선택 가능

- 기존에 사용하던 파일인 경우, 기존 데이터에 맞게 자동 선택됨


step 2. 다운로드


case A. 전체 종목(코스피 + 코스닥) 다운로드.

- [전체 다운로드] 클릭


case B. 종목 검색하여 다운로드.

- [종목 필터] 검색창에 종목코드/명 입력 후 [Enter]

- [검색 결과만 다운로드] 클릭


case C. 미리 생성해놓은 '종목리스트'에 대해 다운로드.

- 종목리스트 경로 입력 후 [연결] 버튼 클릭

- [검색 결과만 다운로드] 클릭
